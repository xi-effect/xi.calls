# @xipkg/calls-config

Статический конфиг для calls-пакетов: сетка видео, onboarding IDs, baseline audio options.

Env-зависимые настройки передаются через `CallsRuntimeConfigProvider` из `@xipkg/calls-providers`.

См. [docs/migrations/calls-runtime-config.md](../../docs/migrations/calls-runtime-config.md)
